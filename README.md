# IRIS-V: Assistive Blink-to-Speech & IoT System 👁️⚡

**IRIS-V** — это автономное устройство, предназначенное для пациентов с тяжелыми нарушениями моторных функций. Оно переводит осознанные паттерны моргания в синтезированную речь и отправляет экстренные IoT-уведомления через Telegram.

---

## 🛠 Технические характеристики (Technical Specs)

* **Microcontroller:** Dual-core **ESP32** (использование FreeRTOS для параллельных задач).
* **Sensors:** Инфракрасные датчики **TCRT5000** для фиксации движений век.
* **Audio:** Модуль **DFPlayer Mini** для автономного воспроизведения речи без ПК.
* **Connectivity:** Wi-Fi интеграция с **Telegram Bot API** для удаленных алертов.

## 🧠 Ключевые разработки (Key Engineering Tasks)

### 1. Adaptive Timing Algorithm (C++)
Я разработал алгоритм адаптивного тайминга, который классифицирует длительность сигналов в диапазоне от **100ms до 2s+**. 
* **Software Debounce Filtering:** Реализована фильтрация для устранения «непроизвольного окулярного шума» (случайных морганий).
* **State Machine:** Прошивка построена на базе конечного автомата для точного определения паттернов (короткое/длинное моргание).

### 2. Hardware Integration
* Интеграция **TCRT5000** через аналоговые/цифровые входы с калибровкой порога чувствительности.
* Настройка **DFPlayer Mini** через UART для мгновенной аудио-обратной связи.

## 👤 Моя роль
* Проектирование архитектуры системы на базе ESP32.
* Написание прошивки на **C++** (Arduino Framework/ESP-IDF).
* Разработка системы уведомлений в Telegram для опекунов.
