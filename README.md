# IRIS-V: Blink-to-Speech Assistive Device 👁️🔊

**IRIS-V** — это устройство, которое переводит паттерны моргания глаз в человеческую речь. Проект разработан для помощи людям с нарушениями двигательных функций и речи.

---

## 🚀 Особенности (Features)
* **Real-time Processing:** Обработка видеопотока с камеры в реальном времени.
* **Blink Detection:** Алгоритм на базе OpenCV/MediaPipe для фиксации морганий.
* **Speech Synthesis:** Преобразование распознанных паттернов в аудио-сообщения.
* **Web Interface:** Удобная панель управления на Flask/Streamlit (тот самый `web_app.py`).

## 🛠 Стек технологий (Tech Stack)
* **Language:** Python
* **Computer Vision:** OpenCV, MediaPipe
* **Web Framework:** Flask / Python-Telegram-Bot
* **Hardware:** Raspberry Pi / Laptop Camera

## 📐 Как это работает
Алгоритм вычисляет **Eye Aspect Ratio (EAR)** для определения закрытия глаз:
$$EAR = \frac{||p_2 - p_6|| + ||p_3 - p_5||}{2||p_1 - p_4||}$$
Если $EAR$ падает ниже порога на определенное время, система регистрирует моргание.

## 📦 Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `.env` и добавьте свой `TELEGRAM_TOKEN`.
4. Запустите: `python web_app.py`.
